Fluid jumbotron

This is a modified jumbotron that occupies the entire horizontal space of its parent. The image placed in the CSS background property is using the zero-quality trick, where the jpg setting is quality: 0, but the width shwon is half the actual pixel dimension. The image is 224k, where the full size would be more than double.
